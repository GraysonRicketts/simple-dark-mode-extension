# Simple Dark Mode Extension

A Chrome extension that toggles dark mode on any page with one click.

## Load Locally

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `simple-dark-mode-extension` directory

The extension icon will appear in the Chrome toolbar.

## Test It

1. Navigate to any webpage
2. Click the extension icon in the toolbar
3. Click **Toggle Dark Mode** to enable dark mode on the current tab
4. Use the **Intensity** slider to adjust the strength (0–100%)
5. Click the button again to turn dark mode off

> Note: The extension requires the `activeTab` and `scripting` permissions, which are granted automatically when loaded unpacked. It will not work on `chrome://` pages or the Chrome Web Store.
